# Mac homebrew

## 显示已安装的软件

```
$ brew list
```

对于一个某个已安装的软件，如 `gcc` ，还可以使用下面的语句来进一步查看安装路径

```
(base) localhost:~ ed$ brew list gcc
/usr/local/Cellar/gcc/9.2.0/bin/c++-9
/usr/local/Cellar/gcc/9.2.0/bin/cpp-9
/usr/local/Cellar/gcc/9.2.0/bin/g++-9

/usr/local/Cellar/gcc/9.2.0/bin/gcc-ar-9
/usr/local/Cellar/gcc/9.2.0/bin/gcc-nm-9
/usr/local/Cellar/gcc/9.2.0/bin/gcc-ranlib-9
```
